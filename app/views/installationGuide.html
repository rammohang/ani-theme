AngularJS Installation.

Installation

  1. Clone this project or Download that ZIP file from the Github repository
  2. Make sure you have bower, gulp and npm installed globally
  3. On the command prompt run the following commands
  4. cd project-directory
  5. bower install
  6. npm install

Run AngularJS

  1. cd project-directory
  2. gulp serve - For development mode
  3. gulp build - concat, minify and generate the files for deployment
  4. gulp serve runs localhost on port 9000, and we can edit gulpfile.js to use different port

Webservices installation.

Installation

  1. Clone this project or Download that ZIP file from the Github repository
  2. Make sure you have eclipse IDE, maven and tomcat server installed
  3. Integrate tomcat server with eclipse to run the project on tomcat.

Run backend services

  1. On the command prompt run the following commands
  2. cd project-directory
  3. mvn -U clean install for development use
  4. Update the eclipse project with maven plugin
  5. mvn package generates war file for production deployment
  6. tomcat runs localhost on 8080 port by default, edit server.xml to use different port

MongoDB Installation.

Installation

  1. Download mongoDB installer and install the software
  2. set mongodb bin folder in path user/system environment variable
  3. create folder structure C:\data\db

Run MongoDB

  1. On the command prompt run the following commands
  2. cd C:\data\db
  3. del mongod.lock
  4. mongod --dbpath C:/data/db
  5. mongoDB runs on 27017 port by default
  6. execute mongo on command prompt to start mongo instance
